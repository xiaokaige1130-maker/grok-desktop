## [0.8.5] — 2026-07-21

**主题：macOS 支持 · 窗口拖拽 · Dock 启动 PATH · 系统菜单**

Release：https://github.com/xiaokaige1130-maker/grok-desktop/releases/tag/v0.8.5

### macOS

| 改动 | 说明 |
|------|------|
| **打包** | `npm run dist:mac` → DMG + ZIP（x64） |
| **标题栏** | `hiddenInset` + 红绿灯位置；顶部拖拽条 / 品牌区 / 顶栏可拖窗口 |
| **系统菜单** | 应用菜单含编辑（Cmd+C/V/A/Z）、查看、窗口 |
| **Dock** | 点击 Dock 图标恢复/前置窗口；About 面板版本号 |
| **CLI 查找** | GUI 从 Dock 启动时 PATH 很短；探测 `~/.grok/bin/grok`、`~/.local/bin`、Homebrew，并给子进程补全 PATH |

### 其它

- `preload` 暴露 `platform`，界面按 `platform-darwin` 适配边距与 ⌘ 快捷键展示  
- 侧栏品牌图标略缩小，适配红绿灯留白  
- 安装包：`Grok-Desktop-0.8.5-macOS-x64.dmg` / `.zip`

## [0.8.4] — 2026-07-20

**主题：会话管理做强 · 右键菜单 · 本地归档/置顶 · Skills/插件设置去重**

Release：https://github.com/xiaokaige1130-maker/grok-desktop/releases/tag/v0.8.4

### 会话右键菜单（侧栏「最近会话」上右键）

| 菜单项 | 说明 |
|--------|------|
| **打开** | 进入该会话 |
| **置顶 / 取消置顶** | 会话出现在侧栏顶部「📌 置顶」分组 |
| **重命名** | 修改会话标题 |
| **导出 Markdown** | 调用官方 `grok export`，弹出另存为 |
| **复制会话 ID** | 复制完整会话 UUID（方便反馈、排查） |
| **复制标题** | 复制当前标题文案 |
| **复制工作目录** | 复制会话关联的 cwd（无目录时禁用） |
| **在文件夹中显示** | 用系统文件管理器打开该会话在 `~/.grok/sessions/` 下的目录 |
| **归档 / 取消归档** | **仅桌面端分组**，不删除 CLI 会话数据；归档出现在底部「📦 归档」 |
| **删除** | 确认后删除会话（并清理本地置顶/归档记录） |

### 侧栏分组

- **置顶** 分组：已置顶会话  
- **按项目** 分组：原有逻辑（工作目录）  
- **归档** 分组：默认折叠；展开状态会记住  

### 设置 · Skills / 插件

- 设置内改为 **摘要列表 +「打开 Skills / 插件 →」**，完整管理放在侧栏独立页  
- 减少设置页与侧栏功能重复  
- Skills 摘要最多展示部分条目，并提示去侧栏看全部  
- 插件仍支持在设置里 **快速安装**（名称 / owner/repo / git URL）

### 安装包

| 文件 | 平台 |
|------|------|
| `Grok-Desktop-0.8.4-Windows-Setup-x64.exe` | Windows 安装版 |
| `Grok-Desktop-0.8.4-Windows-Portable-x64.exe` | Windows 绿色版 |
| `grok-desktop_0.8.4_amd64.deb` | Linux deb |

### 说明

- **归档不是官方 CLI 命令**：数据仍在 `~/.grok/`，只是桌面列表里挪到「归档」组。  
- 置顶 / 归档列表保存在本机桌面配置：`~/.config/linux-grok-desktop/settings.json`（Windows 对应用户配置目录下的 `linux-grok-desktop` / 应用配置）。

## [0.8.3] — 2026-07-20

**主题：对话气泡分色 · 用户与助手一眼可辨**

Release：https://github.com/xiaokaige1130-maker/grok-desktop/releases/tag/v0.8.3

### 对话布局

| 角色 | 样式 |
|------|------|
| **你（用户）** | 靠右 · 紫色系圆角气泡 |

- 工具卡、权限卡、Diff 卡仍占满对话列宽

## [0.8.0] — 2026-07-17 前后

**主题：产品化设置 · 计划按钮 · 权限三档 · 中英 i18n**

### 计划面板

- 「计划」按钮移到**顶栏**（打开会话后始终可见，不再藏在状态条里）  
- 有计划时显示**数量角标**与 **done/total** 进度  
- 快捷键 **`P`** 打开/关闭计划面板  
- 计划面板加宽，条目状态更清晰  

### 权限 / 访问模式（设置 → 权限）

| 模式 | 含义 |
|------|------|
| **审批模式** | 写文件 / 跑命令前需确认 |
| **智能模式**（推荐） | 常规工具自动放行，高风险仍提示 |
| **完全访问** | 工具默认自动批准；可选 YOLO 更激进 |

- 状态条显示当前访问模式徽章  

### 中英界面

- 新增 `renderer/i18n.js`  
- **设置 → 常规 → 界面语言**：中文 / English，即时生效  

### 其它

- 设置页分区与快捷键说明整理  
- 多会话、Diff、搜索、壁纸、通知、更新检查等延续 0.7.x  

### 安装包

- 正式双端：Windows Setup/Portable + Linux deb/AppImage 目标

## [0.7.0] — 更早

**主题：首次引导 · 完成后通知 · 检查更新**

| 能力 | 说明 |
|------|------|
| **首次环境引导** | 启动检测 CLI 是否安装、是否像已登录；可重新检测 / 进入应用 |
| **完成后通知** | 后台会话跑完系统通知（可关；当前会话不打扰） |
| **检查更新** | 启动时查 GitHub Release；有新版本顶栏提示；设置可关 / 手动检查 |
| **快捷键说明** | 设置 → 常规：Enter 排队、引导、N、Ctrl+Tab、`/` 等 |
| **环境诊断** | 设置 → 环境：CLI / 登录状态检测 |
