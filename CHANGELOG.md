## [0.8.8] — 2026-07-22

**主题：Bootstrap UI · 全局浮层 · 官方 CLI 三工作模式（任务 / 计划 / 目标）**

### UI 组件与弹出层

| 改动 | 说明 |
|------|------|
| **Bootstrap 5.3** | 引入 `bootstrap` 依赖；CSS/JS 接入 renderer；打包包含 `node_modules/bootstrap/dist` |
| **计划抽屉** | 执行计划改为 Bootstrap **Offcanvas**（遮罩 / Esc / 关闭）；`scroll:true` 不锁 body；z-index 置顶 |
| **全局滚动条** | 主题化细滚动条覆盖会话列表、对话流、设置、代码块等全部区域 |
| **浮层不占布局** | 斜杠菜单、搜索结果、模型/强度下拉、Goal 条改为绝对/固定浮层，打开时不挤主布局 |

### 工作模式（对齐官方 Grok CLI）

| 模式 | 说明 |
|------|------|
| **任务 (Normal)** | 默认：直接执行与改代码 |
| **计划 (Plan)** | `/plan`：先方案（plan.md）再动手；打开计划抽屉 |
| **目标 (/goal)** | 跨轮目标；输入框**无需**手写 `/goal`，发送时底层自动包装；气泡显示用户原文 |

Composer：附加文件 `@` → 圆形 **`+`**；模式芯片在左侧（任务灰 / 计划琥珀 / 目标紫）。

### 安装包

| 文件 | 平台 |
|------|------|
| `Grok-Desktop-0.8.8-Windows-Portable-x64.exe` | Windows 绿色版 |
| `Grok-Desktop-0.8.8-Windows-Setup-x64.exe` | Windows 安装版 |

### 说明

- 仍依赖本机官方 **Grok CLI**。
- 本版以 Windows 打包为主；macOS / Linux 共用代码。

## [0.8.7] — 2026-07-21

**主题：Windows 原生桌面体验 · 托盘/通知/窗口记忆 · 顶栏工具栏统一**

Release：https://github.com/xiaokaige1130-maker/grok-desktop/releases/tag/v0.8.7

### Windows 桌面壳

| 改动 | 说明 |
|------|------|
| **系统托盘** | 常驻托盘；右键：显示窗口 / 新建会话 / 设置 / 退出；tooltip 显示运行中数量 |
| **关闭 → 托盘** | 点 × 默认不退出（Win/Linux）；托盘退出、菜单退出或 **Ctrl+Q** 才真正退出 |
| **首次托盘提示** | 第一次藏到托盘时系统通知 + 应用内 toast，避免误以为闪退 |
| **最小化到托盘** | 设置可选（需先开启关闭到托盘） |
| **开机自启** | 设置可选，同步 OS login item |
| **完成后通知** | 窗口失焦 / 托盘 / 后台会话都会通知；**点击通知回到对应会话** |
| **任务栏** | 运行中不确定进度条；后台完成时闪烁任务栏 |
| **窗口记忆** | 记住大小、位置、最大化状态 |
| **应用菜单** | 文件 / 编辑 / 查看 / 帮助（新建、设置、检查更新、打开 Releases） |
| **快捷键** | `Ctrl+N` 新建 · `Ctrl+,` 设置 · `Ctrl+P` 计划 · `Ctrl+W` 关标签 · `Ctrl+Tab` 切换 · `Ctrl+1…9` · `Ctrl+Q` 退出 |
| **单实例** | 再次启动会唤起已有窗口 |
| **深色系统主题** | `nativeTheme` 与应用内深色 UI 更一致 |

### 顶栏 UI

| 改动 | 说明 |
|------|------|
| **统一工具栏** | **计划 · 导出 · 重命名 · 删除** 同一高度/圆角/图标/hover |
| **去掉重复计划** | 状态条上的第二个「计划」已删除，只保留右上角 |
| **计划状态** | 打开时高亮；有条目时显示数字角标 |

### 安装包

| 文件 | 平台 |
|------|------|
| `Grok-Desktop-0.8.7-Windows-Portable-x64.exe` | Windows 绿色版 |
| `Grok-Desktop-0.8.7-Windows-Setup-x64.exe` | Windows 安装版（`npm run dist:win:setup`） |

### 说明

- 仍依赖本机官方 **Grok CLI** 登录与额度。
- 未签名构建可能被 SmartScreen 拦截（社区包常见）。
- macOS / Linux 共用代码；本版以 Windows 体验为主，**macOS 0.8.5 能力保留**（hiddenInset、Dock PATH、打包脚本）。

## [0.8.4] — 2026-07-20

**主题：会话管理做强 · 右键菜单 · 本地归档/置顶 · Skills/插件设置去重**

Release：https://github.com/xiaokaige1130-maker/grok-desktop/releases/tag/v0.8.4

### 会话右键菜单（侧栏「最近会话」上右键）

| 菜单项 | 说明 |
|--------|------|
| **打开** | 进入该会话 |
| **置顶 / 取消置顶** | 会话出现在侧栏顶部「📌 置顶」分组 |
| **重命名** | 修改会话标题 |
| **导出 Markdown** | 调用官方 `grok export`，弹出另存为 |
| **复制会话 ID** | 复制完整会话 UUID（方便反馈、排查） |
| **复制标题** | 复制当前标题文案 |
| **复制工作目录** | 复制会话关联的 cwd（无目录时禁用） |
| **在文件夹中显示** | 用系统文件管理器打开该会话在 `~/.grok/sessions/` 下的目录 |
| **归档 / 取消归档** | **仅桌面端分组**，不删除 CLI 会话数据；归档出现在底部「📦 归档」 |
| **删除** | 确认后删除会话（并清理本地置顶/归档记录） |

### 侧栏分组

- **置顶** 分组：已置顶会话  
- **按项目** 分组：原有逻辑（工作目录）  
- **归档** 分组：默认折叠；展开状态会记住  

### 设置 · Skills / 插件

- 设置内改为 **摘要列表 +「打开 Skills / 插件 →」**，完整管理放在侧栏独立页  
- 减少设置页与侧栏功能重复  
- Skills 摘要最多展示部分条目，并提示去侧栏看全部  
- 插件仍支持在设置里 **快速安装**（名称 / owner/repo / git URL）

### 安装包

| 文件 | 平台 |
|------|------|
| `Grok-Desktop-0.8.4-Windows-Setup-x64.exe` | Windows 安装版 |
| `Grok-Desktop-0.8.4-Windows-Portable-x64.exe` | Windows 绿色版 |
| `grok-desktop_0.8.4_amd64.deb` | Linux deb |

### 说明

- **归档不是官方 CLI 命令**：数据仍在 `~/.grok/`，只是桌面列表里挪到「归档」组。  
- 置顶 / 归档列表保存在本机桌面配置：`~/.config/linux-grok-desktop/settings.json`（Windows 对应用户配置目录下的 `linux-grok-desktop` / 应用配置）。
